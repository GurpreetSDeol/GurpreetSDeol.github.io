Projects

  • Maritime IUU Fishing Detection using Machine Learning
    Web Application | GitHub

    Fetched raw AIS positional pings directly from the Global Fishing Watch API and constructed rolling 2-hour windows per vessel track. Engineered physics-based features including speed variance, turning rate and course deviation, then labelled windows using temporal overlap with confirmed GFW fishing events. Trained a LightGBM classifier with calibrated probability outputs, scoring vessel activity against IUCN-tiered marine protected area boundaries to flag potential IUU fishing. Built an interactive Streamlit application to visualise vessel tracks, feature distributions and risk scores.

  • Public Health & Air Quality Dashboard
    Live Dashboard | GitHub

    Built an end-to-end ETL pipeline collecting pollution, weather and health data across major global cities. Automated ingestion using Python APIs, Docker and PostgreSQL and developed an interactive Streamlit dashboard with geospatial visualisations and time-series analysis. Integrated WHO datasets to explore relationships between air pollution exposure and mortality.

  • MSc Dissertation: Football Tactical Analysis with Machine Learning
    Download Thesis

    Analysed 13 seasons of football data using Principal Component Analysis and K-Means clustering to identify tactical evolution across Europe's top leagues. Built a dataset with over 2000 team-season records and 100+ performance metrics, revealing a shift towards possession-based play.

  • BSc Physics Project: Simulation of Electrical Resistance
    Download Thesis

    Simulated electrical resistance in two-dimensional conductive geometries using Python and numerical methods. Modelled electric potential fields using the finite difference method to analyse how geometry and boundary conditions influence current flow.
